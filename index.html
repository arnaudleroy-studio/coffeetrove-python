coffeetrove

Python SDK for CoffeeTrove.com. Access 440,000+ cafes worldwide, 15 coffee origins, 15 brewing methods, 23 bean varieties, 17 drink types, and more.

Python 3.9+ MIT License 440K+ cafes 102 knowledge entries
$ pip install coffeetrove

Installation

Install the base package from PyPI:

pip install coffeetrove

For pandas DataFrame support, install with the optional dependency:

pip install coffeetrove[pandas]

Requires Python 3.9 or later. The only runtime dependency is httpx (for cafe search). All knowledge data (origins, methods, beans, drinks, equipment, comparisons) is bundled with the package and available offline.

Quick Start

import coffeetrove

# Search 440,000+ cafes worldwide (requires internet)
paris_cafes = coffeetrove.cafes("Paris", limit=5)
print(paris_cafes[0]["name"])

# Search cafes by name
results = coffeetrove.search_cafes("Blue Bottle", city="Tokyo")

# Coffee origins with flavor profiles (bundled, works offline)
origins = coffeetrove.origins()
ethiopia = [o for o in origins if o["slug"] == "ethiopia-yirgacheffe"][0]
print(ethiopia["flavor_notes"])  # floral, citrus, blueberry, bergamot

# Brewing methods with guides (bundled, works offline)
methods = coffeetrove.methods()

# Coffee drinks with nutritional info
drinks = coffeetrove.drinks()

# Bean varieties from around the world
beans = coffeetrove.beans()

# Equipment and gear
gear = coffeetrove.equipment()

Using the Client

For more control over HTTP settings and multiple requests, use CoffeeTroveClient as a context manager:

from coffeetrove import CoffeeTroveClient

with CoffeeTroveClient() as client:
    # Search cafes with filters
    cafes = client.cafes(city="Melbourne", limit=20)

    # Get cafes by country
    japan_cafes = client.cafes(country="japan", limit=50)

    # Get a single cafe by its URL slug
    cafe = client.cafe("cafe-de-flore-paris")

    # Access bundled knowledge data
    origins = client.origins()
    methods = client.methods()
    comparisons = client.comparisons()

You can also configure a custom timeout:

client = CoffeeTroveClient(timeout=60.0)

Pandas Support

Every data function accepts an as_dataframe=True parameter to return a pandas DataFrame instead of a list of dicts:

import coffeetrove

# Cafe search results as a DataFrame
df = coffeetrove.cafes("London", limit=100, as_dataframe=True)
print(df[["name", "city", "score"]].describe())

# Origins as a DataFrame
origins_df = coffeetrove.origins(as_dataframe=True)
print(origins_df[["name", "altitude_range", "country_code"]])

Requires the pandas extra: pip install coffeetrove[pandas]

Module Functions

All functions are available directly on the coffeetrove module.

coffeetrove.cafes(city=None, *, country=None, query=None, limit=20, as_dataframe=False)

Requires Internet

Search 440,000+ cafes worldwide. Returns a list of cafe dicts sorted by Golden Drop score.

Parameters:

  • city (str, optional) — Filter by city name.
  • country (str, optional) — Filter by country slug (e.g., "france", "japan").
  • query (str, optional) — Search by cafe name.
  • limit (int) — Maximum results. Default 20, maximum 100.
  • as_dataframe (bool) — Return a pandas DataFrame.
coffeetrove.origins(*, as_dataframe=False)

Offline / Bundled

15 coffee-growing origins with flavor profiles, altitude ranges, and regions. Includes Ethiopia Yirgacheffe, Colombia Huila, Kenya AA, Panama Geisha, and more.

See the full Coffee Origins guide on CoffeeTrove.com.

coffeetrove.methods(*, as_dataframe=False)

Offline / Bundled

15 brewing methods with brew time, difficulty rating, and required equipment. From AeroPress to Turkish to Cold Brew.

See the full Brewing Guides on CoffeeTrove.com.

coffeetrove.drinks(*, as_dataframe=False)

Offline / Bundled

CoffeeTroveClient

The client class provides the same functionality as the module-level functions, plus direct cafe lookup by slug. It manages an HTTP connection pool for efficient multiple requests.

class coffeetrove.CoffeeTroveClient(timeout=30.0)

Create a new client instance. Use as a context manager for automatic cleanup.

Methods:

  • cafes(city, country, query, limit) — Search cafes.
  • cafe(slug) — Get a single cafe by URL slug. Returns None if not found.
  • origins() — Bundled origin data.
  • methods() — Bundled brewing method data.
  • drinks() — Bundled drink data.
  • beans() — Bundled bean data.
  • equipment() — Bundled equipment data.
  • comparisons() — Bundled comparison data.

Static helpers:

  • CoffeeTroveClient.cafe_url(slug) — Build a full CoffeeTrove URL for a cafe.
  • CoffeeTroveClient.origin_url(slug) — Build a full CoffeeTrove URL for an origin.
  • CoffeeTroveClient.method_url(slug) — Build a full CoffeeTrove URL for a method.

Search Functions

coffeetrove.search_cafes(query, *, city=None, country=None, limit=20)

Requires Internet

Search cafes by name. A convenience wrapper around cafes() with the query parameter required.

coffeetrove.cafe(slug)

Requires Internet

Look up a single cafe by its CoffeeTrove URL slug. Returns a dict or None if the slug is not found.

Golden Drop Score

Every cafe in the CoffeeTrove database has a Golden Drop score from 0 to 100. The score is computed from data completeness, review signals, and chain classification. Independent cafes receive a +10 point bonus over chains.

TierScore RangeDescription
Exceptional90 – 100Outstanding across all signals
Excellent80 – 89Well-reviewed, complete data
Notable70 – 79Good data, positive indicators
Common0 – 69Limited data or chain location
